플라스크 시작코드

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
   return 'This is Home!'

if __name__ == '__main__':  
   app.run('0.0.0.0',port=5000,debug=True)

하고 나서 브라우저 창에 http://localhost:5000/ 로 접속
http://localhost: = 내가 만든 항구에
5000 = 5000번 문으로 들어가겠다

5000은 포트라 불린다
변경이 가능하지만 에러가 생길 수 있으니 변경 ㄴㄴ
